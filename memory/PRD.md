# Zitex - منصة الإبداع بالذكاء الاصطناعي

## Problem Statement
منصة متكاملة لإنشاء المواقع وتوليد الصور والفيديوهات باستخدام الذكاء الاصطناعي، مع نظام اشتراكات ونقاط ودعم الدفع البنكي السعودي و PayPal.

## Architecture
- **Frontend**: React 19 + Tailwind CSS + Shadcn UI
- **Backend**: FastAPI (Python)
- **Database**: MongoDB
- **AI Integration**: 
  - GPT-5.2 (النصوص والاقتراحات)
  - Gemini Nano Banana (توليد الصور)
  - Sora 2 (توليد الفيديو - مهيأ)
- **Notifications**: WhatsApp via CallMeBot API

## User Personas
1. **المالك (Owner)**: صلاحيات كاملة مجانية، إدارة الموقع، إشعارات واتساب
2. **الأدمن**: إدارة الطلبات والمدفوعات والعملاء
3. **العميل**: إنشاء مواقع، توليد صور/فيديو بالاشتراك أو الدفع الفردي

### Backend APIs
- `/api/auth/*` - تسجيل، دخول، معلومات المستخدم
- `/api/requests/*` - إنشاء وإدارة طلبات المواقع (+ تجربة مجانية)
- `/api/payments/*` - إدارة المدفوعات + إشعارات واتساب
- `/api/websites/*` - إدارة المواقع المنجزة
- `/api/generate/image` - توليد الصور (+ دعم المجاني)
- `/api/generate/video` - توليد الفيديو (+ دعم المجاني)
- `/api/pricing` - عرض الأسعار
- `/api/admin/*` - إحصائيات، مستخدمين، إعدادات

### Frontend Pages
- `/` - الصفحة الرئيسية
- `/login`, `/register` - المصادقة
- `/pricing` - الأسعار والباقات
- `/dashboard/*` - لوحة تحكم العميل (+ عرض التجارب المجانية)
- `/dashboard/images` - توليد الصور (+ بانر المجاني)
- `/dashboard/videos` - توليد الفيديو (+ بانر المجاني)
- `/admin/*` - لوحة تحكم الأدمن
- `/admin/settings` - إعدادات الدفع + واتساب
